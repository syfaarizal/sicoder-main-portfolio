üîÄ Conditions: if, else if, else

Logic in JavaScript starts with one simple question:

If this is true, what should happen? If not, then what?

JavaScript 

if (condition) {
  // runs if true
} else {
  // runs if false
}

Real-life example:

JavaScript 

let age = 18;

if (age >= 17) {
  console.log("You may enter the cinema üé¨");
} else {
  console.log("Sorry, not old enough üòÖ");
}

JavaScript has zero empathy. True runs. False gets skipped.

üß≠ Switch Case

When conditions get messy, switch keeps things readable.

JavaScript

  case "Monday":
    console.log("Time to work üíº");
    break;
  case "Sunday":
    console.log("Rest day üò¥");
    break;
  default:
    console.log("Just another day");

Forget break? Everything below it runs. Chaos mode activated.

üîÅ Loops: Repeat Without Crying

for loop

JavaScript

  console.log("Count:", i);
}

while loop

JavaScript 

let count = 1;

while (count <= 3) {
  console.log(count);
  count++;
}

do while loop

JavaScript 

let num = 1;

do {
  console.log(num);
  num++;
} while (num <= 3);
