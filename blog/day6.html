🔢 Day 6: Odd, Even, or Special?

Yo! Back again with day 6 challenge. Today I'm playing with numbers, but not just checking odd/even. I added some layered logic to sharpen my coding reasoning 🧠✨

🎯 Challenge Goals:

  • Practice nested conditions (if, else if, else)
  • Understand odd, even, and multiple concepts
  • Add input validation & more human-friendly output

📝 Program Description:

  1. User inputs a number
  2. Program checks if the number is:
    • ✅ Odd
    • ✅ Even
    • 🎉 Multiple of 10 (shows special message)
  3. Output includes emojis to make it more expressive 😄

Example Output:


Enter a number: 27
➡️ 27 is an odd number 🔹

Enter a number: 44
➡️ 44 is an even number 🔸

Enter a number: 30
🎉 30 is an even number & multiple of 10!

🛠️ Tools & Concepts:

  • prompt() and parseInt() to get user input
  • isNaN() for input validation
  • if and else if for conditional logic
  • % (modulus) operator to check odd/even/multiples

⚙️ Basic Implementation:


const input = parseInt(prompt("Enter a number dude!:"));

if (isNaN(input)) {
    console.log("Lol that's not a number 😅");
} else if (input % 2 === 0) {
    console.log("This is an even number dude! 👍");
} else {
    console.log("This is an odd number dude! 👌");
}

✅ Full Code Version 1:


const input = parseInt(prompt("Enter a number dude:"));

if (isNaN(input)) {
    console.log("Lol that's not a number 😅"); 
} else if (input % 10 === 0 && input % 2 === 0) {
    console.log("This is an even number and multiple of 10 dude! 🎉");
} else if (input % 2 === 0) {
    console.log("This is an even number dude! 👍");
} else {
    console.log("This is an odd number dude! 👌");
}

🔁 Bonus Challenge: Loop Version


let input;

while (true) {
    input = prompt("Enter a number dude! (type 'exit' to quit)");

    if (input === "exit") {
        console.log("Alright dude, exiting the program. Bye 👋");
        break;
    }
    const number = parseFloat(input);

    if (isNaN(number)) {
        console.log("Lol that's not a number 😅"); 
    } else if (number % 10 === 0 && number % 2 === 0) {
        console.log("This is an even number and multiple of 10 dude! 🎉");
    } else if (number % 2 === 0) {
        console.log("This is an even number dude! 🔸");
    } else {
        console.log("This is an odd number dude! 🔹");
    }
}

📉 Addition: Check Negative Numbers

🚀 Final Code:


let input;

while (true) {
    input = prompt("Enter a number dude! (type 'exit' to quit)");

    if (input === "exit") {
        console.log("Alright dude, exiting the program. Bye 👋");
        break;
    }

    const number = parseFloat(input);

    if (isNaN(number)) {
        console.log("❌ Lol that's not a number 😅"); 
    } else if (number < 0) {
        console.log("📉 This is a negative number...");
    } else if (number % 10 === 0 && number % 2 === 0) {
        console.log("🎊 This is an even number and multiple of 10 dude!");
    } else if (number % 2 === 0) {
        console.log("🔸 This is an even number dude!");
    } else {
        console.log("🔹 This is an odd number dude!");
    }
}
