‚úÇÔ∏è splice() vs üç∞ slice()

These two are often confused ‚Äî but very different.

JavaScript 

// splice() changes the original array
let numbers = [1, 2, 3, 4];
numbers.splice(1, 2);
// result: [1, 4]

// slice() copies only
let nums = [1, 2, 3, 4];
let newNums = nums.slice(1, 3);
// result: [2, 3]

Easy rule:

  • splice = brutal
  • slice = polite

üß† Modern Array Power: map() & filter()

map() ‚Äî transform data

filter() ‚Äî select data

JavaScript

Real projects use these every single day.
