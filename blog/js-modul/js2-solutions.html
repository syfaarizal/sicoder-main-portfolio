üî• Challenge 1 ‚Äî FizzBuzz

Rules recap:

  • Multiple of 3 ‚Üí Fizz
  • Multiple of 5 ‚Üí Buzz
  • Multiple of 3 & 5 ‚Üí FizzBuzz
JavaScript 

let number = 15;

if (number % 3 === 0 && number % 5 === 0) {
  console.log("FizzBuzz");
} else if (number % 3 === 0) {
  console.log("Fizz");
} else if (number % 5 === 0) {
  console.log("Buzz");
} else {
  console.log(number);
}

Why this order matters:
The combined condition must be checked first. Otherwise, JavaScript will exit early and your logic breaks.
