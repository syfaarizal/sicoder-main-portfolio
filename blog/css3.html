CSS has the position property to control how elements "stick" to the page.

Type Brief Description
staticDefault (follows HTML flow)

z-index: Who's On Top?

z-index controls the stacking order of elements (z-axis). Higher value = more forward.

CSS 

.card {
  position: absolute;
  z-index: 10;
}

Useful so popups/modals don't get covered by other content. Think of it like Photoshop layers üé®

üßò Flexbox: Stress-Free Layouts & Centering

üîß Important container properties:

CSS 

.container {
  display: flex;
  justify-content: center;   /* Horizontal alignment */
  align-items: center;       /* Vertical alignment */
  flex-direction: row;       /* Can be 'row' or 'column' */
}

üîß Important item properties:

CSS 

.item {
  flex-grow: 1;
}

üß™ Simple Flexbox Example:

CSS 

.flexbox {
  display: flex;
  gap: 20px;
  justify-content: space-between;
}
HTML 

<div class="flexbox">
  <div class="box">1</div>
  <div class="box">2</div>
  <div class="box">3</div>
</div>
Property Function
justify-contentHorizontal position (start, center, end, space-between)
align-itemsVertical position (start, center, stretch, end)
flex-directionLayout order: row / column

üß© CSS Grid: For More Complex Layouts

Grid is like Flexbox's "matrix" version ‚Äî perfect for 2D layouts (rows & columns).

üîß Basic example:

CSS

  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 16px;
}

üîß Important properties:

Property Function
grid-template-columnsNumber & width of columns
grid-template-rowsNumber & height of rows
gapSpace between elements
grid-column / rowFor merging columns/rows (span)

üß™ CSS Grid Example:

HTML 

<div class="grid-container">
  <div class="grid-item">A</div>
  <div class="grid-item">B</div>
  <div class="grid-item">C</div>
</div>
CSS 

.grid-container {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr;
}

üß† 1fr = 1 fraction of available space. Think of it as: 1 slot out of 3 equally divided slots.
